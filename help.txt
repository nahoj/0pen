Usage: 0pen (option|path)*
Ex: 0pen -m toto -f toto/africa.ogg
  --input-list FILE Use the given file list instead of browsing the filesystem
  -i                
  --first FILE      First file to open
  -f                
  --more            Don't exclude files starting with +, only ._~
  -+                
  -n N              Open/output at most N files
  --alpha           Read all files once in file-manager order, i.e., in lexical order except '9' < '10'
  -a                
  --pick MODE       Pick files according to the given pick mode
  -p                
  --raw             Read all files once without ordering or shuffling them
  -r                
  --shuffle         (default) Shuffle all files then loop through them
  --half-life DAYS  (float) Decrease file weight since creation/modification (default = 0. = don't)
  --series N        Max length of a number series (default = 1)

  --list            Print a list of files, don't open them or interact
  -l                
  --command string  Command with which to open files
  -c                
  --mplayer         (default)
  --vlc             
  --xdg             (xdg-open)
  --break DELAY     Delay to interact between playing two files, in seconds (default = 0.7)
  -b                
  --music           Only play audio even for video files (MPlayer only)
  -m                
  --not-music       Don't do that (default = follow main file type)
  -M                
  -V                Shift video colors (MPlayer only)
  -help             Display this list of options
  --help            Display this list of options
